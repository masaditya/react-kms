Nama Kelompok : Rafqi Agatha, Aditya Eka

## :book: Open BO dulu kak!

clone repo ini

```bash
git clone https://github.com/masaditya/react-kms.git
```

masuk ke directory project

```bash
cd react-kms
```

Jalankan command

```bash
npm install
```

Lalu run server dengan command

```bash
npm start
```
